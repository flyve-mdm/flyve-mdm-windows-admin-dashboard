Features


Administration console
You can manage your company’s mobile fleet simply and easily, via a mobile application. Flyve MDM provides access to a centralized mobile device management tool, which offers the following features:
  • Intuitive interface
  • Users’ accounts configuration and management tool
  • Installation / removal of applications
  • Ability to control and monitor the security policy
